페이지가 로드되면 '환영합니다.'라는 경고창을 띄운다.

버튼을 클릭하면 '버튼을 클릭하셨습니다.'라는 경고창을 띄운다.


버튼에 마우스를 오버하면 '마우스를 버튼 위에 올리셨어요.'라는 경고창을 띄운다.


버튼을 클릭하면 '삭제하시겠습니까?' 확인/취소창을 띄운다.
확인을 클릭하면 '삭제완료', 취소를 클릭하면 '삭제취소'라는 경고창을 띄운다.


버튼을 클릭하면 'e-mail을 입력하세요'라는 프롬프트창을 띄운다.
확인을 클릭하면 입력한 e-mail+'이 등록되었습니다.'라는 경고창을 띄운다.
취소를 클릭하면 'e-mail 등록이 취소되었습니다.'라는 경고창을 띄운다.


버튼을 클릭하면 '이 페이지를 출력할까요?'라는 확인/취소 창을 띄운다.
확인을 클릭하면 인쇄창을 띄우고, 취소를 클릭하면 '인쇄취소'라는 경고창을 띄운다.


버튼을 클릭하면 popup.html파일을 팝업창으로 띄운다.
