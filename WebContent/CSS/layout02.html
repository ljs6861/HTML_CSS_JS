HTM5와 CSS3를 이용한 웹 사이트 개발

Article에 해당하는 내용이 들어갑니다.

HTML5와 CSS3를 이용해서 다양한 웹 페이지 레이아웃을 구성해 보겠습니다.
웹 페이지의 레이아웃은 일정한 패턴이 있습니다.
따라서 CSS를 이용해서 레이아웃을 잡는 연습을 많이 하게 되면, 사이트를 새로 만들거나 유지보수시 매우 편리합니다.

Section 내용입니다.

HTML5의 시멘틱 태그는 다음과 같습니다.
header, footer, nav, article, section, aside, hgroup
현재 웹 페이지에 대한 권리는 모두에게 공개되었습니다.
